# バックエンド実装ガイド（技術書）

このドキュメントは、FastAPI / Python / Redis / DDD / ヘキサゴナルアーキテクチャに不慣れなエンジニア向けに、バックエンド実装の読み方を技術書レベルで丁寧に解説するものです。コードを読む前に「全体像→概念→対応付け→主要フロー→詳細」の順で理解できるように構成しています。

## 対象読者

- Python/FastAPI/Redis/DDD/ヘキサゴナルに詳しくないエンジニア
- 既存コードの保守・拡張に入る必要がある人
- アーキテクチャの意図を理解し、正しい修正をしたい人

## 前提知識（最低限）

- HTTP の基本（GET/POST）
- SQL の基本（SELECT/INSERT/UPDATE）
- 非同期処理の概念（I/O 待ちと並行処理の違い）
